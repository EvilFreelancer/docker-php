# Docker PHP+NGINX generator

Small collection of scripts and configs for begin.

    You need Docker Compose at least 1.17 version

## How to use

Copy the ENV example

    cp .env.example .env

Start the build

    docker-compose bild

Run the image

    docker-compose up -d
